# CreatoCrew Temporary Website (GitHub Pages)

## What this is
A one-page temporary launch site (index.html) + asset folders, ready for GitHub Pages.

## Folder structure
- index.html
- assets/brand/logo.png
- assets/hero/hero.png
- assets/clients/*.png
- assets/projects/*.png
- docs/brochure.pdf

## Replace placeholders
1. Replace client logos in `assets/clients/`
2. Replace project thumbnails in `assets/projects/`
3. Edit Instagram links + titles in `index.html` (search for POST_ID_)

## GitHub Pages steps (quick)
1. Create a GitHub repo (public).
2. Upload these files/folders (or drag-drop).
3. Go to Settings → Pages
   - Source: Deploy from a branch
   - Branch: main / root
4. Your site URL will appear in Pages.

## Custom domain (optional)
- Add your domain in Settings → Pages → Custom domain.
- Update DNS at your domain provider as per GitHub Pages instructions.
